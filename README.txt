ΕΡΓΑΣΙΑ C++ 2020

CONTROLS: ARROW KEYS/WASD to move, SPACE to shoot, (G).

Μικρός μου άρεσε πολύ το Tyrian2000 οπότε σκέφτηκα να κάνω κάτι σαν αυτό και απέτυχα :P, οπότε αυτό που φτιάξαμε είναι το potatoTyrian2020.

Σκοπός: Προσπαθείς να ξεφύγεις από τους αντίπαλους. Σε κάθε επίπεδο πρέπει να σκοτώσεις έναν συγκεκριμένο αριθμό αντιπάλων για να ξεφύγεις. Το παιχνίδι έχει 5 επίπεδα.

Το σκάφος σου μπορεί να δεχθεί 3 βολές μέχρι να πεθάνει.
Τα αντίπαλα σκάφη μπορούν επίσης να δεχθούν 3 βολές μεχρι να πεθάνουν.

Πατώντας G μπαίνεις σε "Godmode" για το τρέχον επίπεδο και δεν πεθαίνεις όσες βολές κι αν σε πετύχουν.
